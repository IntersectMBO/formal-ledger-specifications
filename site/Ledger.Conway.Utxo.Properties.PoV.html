We can express this concisely as follows: $$\mbox{\AgdaField{getCoin}~\ab{s}} + \mbox{\AgdaField{getCoin}~\ab{txwdrls}} \mbox{\AgdaFunction{}} (\mbox{\ab{tx}~.\AgdaField{isValid}}) \mbox{\AgdaField{getCoin} \ab{s'}},$$ where $~:~\AgdaDatatype{Bool}~~{0, 1}$ is the *characteristic function*, which returns 0 for false and 1 for true. - *Formally*. 
    UTXOpov : { : UTxOEnv} {tx : Tx} {s s' : UTxOState}
   txidOf tx  map proj (dom (UTxOOf s))
     s  tx ,UTXO s'
   getCoin s + getCoin (wdrlsOf tx) *  (tx .isValid)  getCoin s'
