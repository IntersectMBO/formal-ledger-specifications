    Theorem (The CERTS rule preserves value).

    Informally.

    Let l be a list of DCerts, and let s, s be CertStates such that s l ,CERTS s. Then, the value of s is equal to the value of s plus the value of the withdrawals in .

    Formally.

        CERTS-pov : { : CertEnv} {s s  : CertState}
       [ a  dom (WithdrawalsOf ) ] NetworkIdOf a  NetworkId
         s  l ,CERTS s
       getCoin s  getCoin s + getCoin (WithdrawalsOf )

    Proof.

        CERTS-pov { = } validNetId (run (pre-cert , certs)) =
      trans  (PRE-CERT-pov validNetId pre-cert)
             (cong (_+ getCoin (WithdrawalsOf )) (sts-pov certs))
