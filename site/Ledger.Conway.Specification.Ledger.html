    Ledger¶

    This section is part of the Ledger.Conway.Specification.Ledger module of the formal ledger specification, where the entire state transformation of the ledger state caused by a valid transaction can now be given as a combination of the previously defined transition systems.

    Conway specifics

    As there is nothing new to the Conway era in this part of the ledger, we do not present any details of the Agda formalization.

    {-# OPTIONS --safe #-}

    txgov : TxBody  List (GovVote  GovProposal)
txgov txb = map inj txGovProposals ++ map inj txGovVotes
  where open TxBody txb

rmOrphanDRepVotes : CertState  GovState  GovState
rmOrphanDRepVotes cs govSt = L.map (map go) govSt
  where
   ifDRepRegistered : Voter  Type
   ifDRepRegistered (r , c) = r  DRep  c  dom (DRepsOf cs)

   go : GovActionState  GovActionState
   go gas = record gas { votes = filterKeys ifDRepRegistered (gas .votes) }

allColdCreds : GovState  EnactState   Credential
allColdCreds govSt es =
  ccCreds (es .cc)  concatMap ( (_ , st)  proposedCC (GovActionOf st)) (fromList govSt)
